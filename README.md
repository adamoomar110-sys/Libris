# Libris

Libris is a digital library and document management system.

## Features
- **Document Indexing**: Powerful search and indexing capabilities for digital documents.
- **Web Interface**: Clean and responsive UI for managing your library.
- **Cross-Platform**: Accessible via web and mobile modes.

## Getting Started

### Prerequisites
- Python 3.12+
- Node.js & npm

### Installation
1. Install backend dependencies:
   ```bash
   cd backend
   pip install -r requirements.txt
   ```
2. Install frontend dependencies:
   ```bash
   cd frontend
   npm install
   ```

### Running the App
Use the `start_app.py` script:
```bash
python start_app.py --mode web
```
